#   AI海龟汤游戏 PRD

##  产品目标
为小朋友提供一个寓教于乐的海龟汤游戏。在游戏过程中，培养小朋友的逻辑思维能力，发散思维能力，创造力。

##  核心功能
### MVP1(必须做)

1.  开始页面
    - 规则介绍
    - 开始游戏按钮
2.  游戏页面
    - 需要一个AI裁判，并具有下列功能：
        - 出题
        - 根据答案和游戏规则回答玩家的问题，智能回答是，不是，不相关
        - 给出提示，引导玩家提出正确的问题
        - 宣布答案
    - 对话框
        - 玩家可以输入答案
        - 展示ai裁判的回答

    - 直接显示答案按钮
        - 玩家可以点击直接看答案
3.  游戏结束页面
    - 如果玩家猜到答案，显示成功页面
    - 如果玩家直接看答案，显示游戏答案
    - 再来一次按钮
4.  语音功能
    - ai裁判可以读出问题
    - ai裁判可以读出答案或引导词
    - 玩家可以通过打字或语音输入问题

5.  异常处理
    - 网络问题，提示用户查看网络重试
    - 技术问题，比如AI连接不上，提示用户重试

### 后续
1.  登录功能
2.  让玩家选择问题难度
3.  记录游戏结果，在中断游戏后，下次可继续上次游戏
4.  给玩家打分，并设置玩家得分排行榜
5.  可以联网跟好朋友一起玩海龟汤游戏
6.  可以一键分享游戏结果到社交媒体如微信/小红书等
7.  玩家可以给其他玩家/指定玩家出题

### 用户交互
- 整体风格：悬疑风格，黑灰色调
- 字体要大，让阅读轻松

## 技术栈建议
- 前端：React + TypeScript + Tailwind CSS
- 后端：Node.js + Express
- AI：DeepSeek / Claude API
- 部署：Vercel
