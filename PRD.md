#   AI海龟汤游戏 PRD

##  产品目标
1.  为小朋友提供一个寓教于乐的海龟汤游戏。在游戏过程中，培养小朋友的逻辑思维能力，发散思维能力，创造力。
2.  游戏内容安全，符合小朋友阅读心理，符合未成年人保护标准，无暴力，色情，惊悚成分

##  核心功能
### MVP1(必须做)

1.  开始页面
    -   规则介绍
    -   开始游戏按钮
2.  游戏页面
    -   
    -   对话框
        -   玩家可以输入答案
        -   展示ai裁判的回答

    -   直接显示答案按钮
        -   玩家可以点击直接看答案
3.  游戏结束页面
    -   游戏结束展示答案，玩家称号，对玩家的评价，对玩家的鼓励
    -   根据玩家表现给玩家称号，如
    - -     20步以内找到答案，称号：小小名侦探
    - -     40步以内找到答案，称号：逻辑小天才
    - -     50步以上或直接看答案，称号：见习小侦探
    -   再来一次按钮

4.  AI裁判
    -   出题，题目需是“儿童友好型”（如：科学常识、日常生活、经典寓言变体）
    -   默认是20步以内可以回答上的简单问题
    -   只能用是，不是，不相关回答玩家问题
    -   从玩家第10个问题开始，给出引导性提示
    -   从玩家第20个问题开始，提供’直接看答案‘的选项
    -   宣布答案，并解释答案的逻辑

5.  内容审核
    -   在展示结果前，接入微信或阿里云的内容安全 API，过滤敏感词。

6.  异常处理
    -   网络问题，提示用户查看网络重试
    -   技术问题，比如AI连接不上，提示用户重试
    -   回答尽量幽默，如果“侦探助理去查资料啦，请稍等”。

### 后续
1.  语音功能（可选迭代）
    -   ai裁判可以读出题目、提示与答案（TTS）
    -   玩家可以通过语音输入问题（STT）
2.  登录功能
3.  让玩家选择问题难度
4.  记录游戏结果，在中断游戏后，下次可继续上次游戏
5.  给玩家打分，并设置玩家得分排行榜
6.  可以联网跟好朋友一起玩海龟汤游戏
7.  可以一键分享游戏结果到社交媒体如微信/小红书等
8.  玩家可以给其他玩家/指定玩家出题，ai辅助完善逻辑

### 用户交互
-   整体风格：侦探事务所风格，小清新配色，画风可爱
-   ai裁判以可爱侦探形象出现（如小兔子警官）
-   字体要大，让阅读轻松
-   采用类似即时通讯的界面，AI 回复带有气泡动画
-   提供“提问范式”按钮（例如：“是不是关于xx的？”、“这个物品是xx吗？”），辅助打字慢的小朋友


## 技术栈建议
- 前端：React + TypeScript + Tailwind CSS
- 后端：Node.js + Express
- AI：DeepSeek / Claude API
- 部署：Vercel
